### Calculadora Java

Este é um simples programa de calculadora implementado em Java. Ele permite que o usuário insira dois números e escolha entre quatro operações matemáticas: adição, subtração, multiplicação e divisão. O programa então executa a operação selecionada e exibe o resultado.

### Como usar

1. Compile o programa usando um compilador Java, como o `javac`.
2. Execute o programa compilado usando o comando `java Calculadora`.
3. Siga as instruções exibidas no console para inserir os números e selecionar a operação desejada.

### Funcionalidades

- **Adição:** Soma os dois números inseridos pelo usuário.
- **Subtração:** Subtrai o segundo número do primeiro.
- **Multiplicação:** Multiplica os dois números inseridos.
- **Divisão:** Divide o primeiro número pelo segundo, desde que o segundo número não seja zero (para evitar a divisão por zero).
